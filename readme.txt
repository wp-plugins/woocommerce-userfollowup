== Description ==

WooCommerce Userfollowup allows you to send follow up emails and SMS to your WooCommerce customers via Userfollowup (http://userfollowup.com/) based on rules you define. Contact automatically your customers to prevent cart abandonment, send special offers to customers that saw certain products, bring them back to your store and more.

== Installation ==

= Automatic =

1. In the admin panel plugins page click Add New
2. Search for WP UserFollowUp
3. Find WP UserFollowUp in the list an click Install Now
4. Click OK when prompted
5. Activate the plugin
6. Done!

= Manual =

1. Extract wp-userfollowup.zip to your `wp-content/plugins` directory.
2. In the admin panel under plugins activate woocommerce-userfollowup.
3. Done!

= Configuration =

1. Visit the newly created menu for woocommerce-userfollowup on your dashboard's Settings menu
2. Enter your userfollowup tracking ID
3. Save your settings
